# 💎 Diamonds Price Prediction

A machine learning project that predicts diamond prices based on various features like carat, cut, color, clarity, depth, and table.
The project includes full data preprocessing, visualization, model training, evaluation, and hyperparameter tuning.

🚀 Project Overview

The goal of this project is to build a regression model that accurately predicts diamond prices.
This project covers the complete ML pipeline from data cleaning to model deployment-ready output.

📊 Features

Data preprocessing (handling missing values, encoding, scaling)

Exploratory Data Analysis (EDA) with Matplotlib & Plotly

Interactive visualizations for better insights

Model training and comparison using multiple regressors

Hyperparameter tuning for optimal performance

🧠 Algorithms Used

Linear Regression

Decision Tree Regressor

Random Forest Regressor

KNN Regressor

SVR

XGBoost Regressor

🧩 Tools & Libraries

Python

Pandas

NumPy

Matplotlib

Plotly

Scikit-learn

XGBoost



⚙️ Results

Compared all models based on R² Score, MAE, and RMSE

Achieved the best accuracy with XGBoost Regressor after tuning
